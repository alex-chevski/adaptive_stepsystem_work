STEP_3D -  интеллектуальный интерфейс для автоматической оценки трудоемкости и стоимости 3D-услуг, подбора оптимальных исполнителей, получения коммерческого предложения и осуществления сделки по публичной оферте в области

{

  • 3D  сканирования

  • Реверсивного инжиниринга (CAD)

  • 3D  моделирования

  • 3D  печати

}
